# 🚀 SosinskiWeb - Personal Portfolio

Nowoczesna strona typu landing page stworzona dla marki osobistej. Projekt skupia się na wysokiej wydajności, czystym kodzie oraz responsywności (Mobile First).

## 🛠️ Stack Techniczny
* **Frontend:** HTML5 (Semantyczny), Tailwind CSS, JavaScript (ES6+)
* **Biblioteki:** Canvas-confetti (interakcja po wysłaniu formularza), FontAwesome
* **Infrastruktura:** Zarządzanie własnym serwerem na LH.pl (obsługa rekordów DNS, certyfikat SSL, konfiguracja FTP/SSH)
* **Wdrożenie:** Custom Code (bez użycia CMS typu WordPress)

## 🌟 Kluczowe Funkcjonalności
* **Dedykowany Formularz Kontaktowy:** Integracja z FormSubmit z obsługą stanów (Loading/Success) oraz efektem konfetti.
* **Smart Service Selection:** System automatycznego wybierania usługi w formularzu po kliknięciu w konkretną ofertę.
* **Optymalizacja Wydajności:** Użycie formatów WebP dla obrazów oraz asynchroniczne ładowanie zasobów (Google PageSpeed Score: ~95+).
* **Responsive Web Design:** Pełne dostosowanie do urządzeń mobilnych.

### Zastosowane techniki optymalizacji:
* **Next-Gen Images:** Wykorzystanie formatu `.webp`, co zredukowało wagę grafik o ponad 70% bez utraty jakości.
* **Critical CSS:** Użycie Tailwind CSS pozwoliło na usunięcie nieużywanych stylów i błyskawiczne renderowanie drzewa DOM.
* **Async & Defer:** Asynchroniczne ładowanie skryptów (FontAwesome, Confetti) zapobiega blokowaniu renderowania strony.
* **Semantic HTML:** Poprawna struktura tagów, co przełożyło się na wynik 100/100 w kategorii SEO i wysoki wskaźnik Accessibility.
